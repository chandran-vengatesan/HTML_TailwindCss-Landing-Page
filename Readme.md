# Learning Landing Page

This is a responsive landing page for a learning platform called "AskMeIdentity". The page is built with HTML and Tailwind CSS.

## Features

- **Responsive Design:** The landing page is fully responsive and works on all devices.
- **Navigation:** A navigation bar with links to different sections of the page.
- **Hero Section:** A hero section with a call to action.
- **Featured Courses:** A section to display featured courses.
- **Why Learn With Us:** A section to highlight the benefits of the platform.
- **Categories:** A section to display different categories of courses.
- **Subscription:** A section to subscribe to the newsletter.
- **Testimonials:** A section to display testimonials from students.
- **Footer:** A footer with links to social media and other information.

## Technologies Used

- HTML
- Tailwind CSS
- Google Fonts

## How to Use

1.  Clone the repository.
2.  Open the `index.html` file in your browser.
